# PJH-RPG

2D 캔버스 기반 RPG 게임 프로젝트. 프레임워크 없이 순수 HTML/CSS/JS로 제작.

## 실행 방법

정적 파일이므로 로컬 서버로 열면 됩니다.

```
python -m http.server 8000
```

그 후 브라우저에서 `http://localhost:8000` 접속.

## 현재 구현

- 타일 기반 맵과 벽 충돌
- 방향키 / WASD로 부드러운 타일 단위 이동
- NPC와의 간단한 대화 상호작용
- 보물 상자(골드 획득)
- HUD (레벨 / HP / 골드)
